### Homework Problem Tracker

# Overview
This local website is a tool to keep track of problem numbers for homework assignments. Problem numbers are selected at random to simulate a test-like environment, where problems of the same type likely won't be presented in a single group. The randomness also promotes the habit of reading the problem's instructions every time, instead of just reading the instructions once then completing the block of problems in one go.

# Accessing the Local Website
- The three files (tracker.html, styles.css, and script.js) need to be downloaded to the same directory on your computer.
- To use the local website, go to your file manager, open tracker.html with your browser.

# Instructions for Use
- Problem Entry
    - You may enter up to five ranges of problems.
    - Problem ranges can be specified as all, only odd, or only even.
    - If you need to add more problems, or have single problems that aren't in a range, you can add individual numbers in a comma seperated format.
- Problem Tracker
    - The progress bar at the top tracks your progress through the assignment.
    - The vertical red line on the progress bar marks the almost finished (80%) point.
    - Pressing "Next Question" will mark the current problem as completed and provide a new randomly selected problem.
    - Pressing "Skip Question" will return the current problem to the pool of uncompleted problems and provide a new randomly selected problem.
    - The timer in the bottom left corner is for measuring problem-solving speed.
